Value attribute in input--We can change the input value





ReadOnly attribute in input-- we can't change the value
whenever we place the cursor on the text box it will show this is active





disabled attribute in input--completely disabled we can't see the hover on the text box





size attribute in input-specify the visible width{not the no of characters}we can use
this size for input type="tel",search,url,email,password,



maxlength attribute in input --specify how many character user have to enter



autofocus attribute in input--used for intimating the user to enter the input to this place
eg:if u open the page it prompt u that mean cursor is blink in particular place to enter the value



formenctype="text/plain" this means not encoding [not commonly used]
send the form data as plain text where name and value pair is seperated by line break
Rarely used [not suitable for file uploads]

formenctype="multipart/form-data" this is typically used for file uploads
it indicates form data will be encoded as a series of parts include binary data as well as regular form fields

formenctype="application/x-www-form-urlencoded"default value used for regular form submissions
It encode the form data in url encoded format
where space replaced with + signs and special characters replaced with percent encoded
